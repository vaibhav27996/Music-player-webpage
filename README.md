# **This is responsive Music player web page which is develped using HTML and CSS**

* On left side of web page , showing the songs list related to that one singer or music composer.That play list is scrollable.


* On right side of web page , showing the songs list related to similar artist and also recently played list.All the play list is scrollable.

- At bottom of web page, there is a music control which has a next,previous,stop ,volume control button etc.
